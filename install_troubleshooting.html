Installation Troubleshooting

Windows
SmartScreen
Right-click installer → Properties → Unblock → Run as Administrator
Signature
Right-click installer → Properties → Digital Signatures → Details
Hash
certutil -hashfile C:\path\to\installer.exe SHA256
macOS
Gatekeeper
System Settings → Privacy & Security → Allow Anyway
Signature
codesign -dv --verbose=2 /Applications/The3rdDemon.app
Hash
shasum -a 256 ~/Downloads/The3rdDemon.dmg
Android
Unknown Sources
Settings → Security → Install unknown apps → Allow
Signature
apksigner verify --print-certs The3rdDemon.apk
Hash
sha256sum The3rdDemon.apk
Rollback
Use previous tag v1.0-final and redeploy
